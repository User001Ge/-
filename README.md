# Election Simulator v2

ეს ვერსია გათვლილია ისეთ არჩევნებზე, სადაც:
- მონაცემებში არის კანდიდატების სრული სია;
- თითო ამომრჩეველს აქვს პრეფერენციის ქულა ყველა კანდიდატზე;
- კონკრეტულ არჩევნებში UI-დან ყოველთვის ირჩევა ზუსტად 3 კანდიდატი.

## ფაილის ფორმატი
Excel ფაილში:
- პირველი სვეტი = ამომრჩევლების სახელები
- პირველი მწკრივი = კანდიდატების სახელები
- შიდა უჯრედები = პრეფერენციის ქულები

მაგალითი:
- A1 = `მღვდელმთავარი`
- B1:... = კანდიდატების სახელები
- A2:A... = ამომრჩევლები
- B2:... = ქულები

## გაშვება ლოკალურად
```bash
pip install -r requirements.txt
streamlit run app.py
```

## არქიტექტურა
- `data_loader.py` — Excel-ის წაკითხვა და ვალიდაცია
- `election_engine.py` — 3-კანდიდატიანი სიმულაციის ბირთვი
- `app.py` — Streamlit ინტერფეისი

## მთავარი იდეა
კოდი არ არის მიბმული კონკრეტულ სახელებზე. თუ კანდიდატების სია ან ქულები შეიცვლება, საკმარისია Excel ფაილის განახლება.
